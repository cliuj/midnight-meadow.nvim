# midnight-meadow

A dark (Neo)vim theme.

## Installation

> With [lazy.nvim](https://github.com/folke/lazy.nvim)
```lua
{
  "cliuj/midnight-meadow.nvim",
}
```

### Usage
###### vimscript
```vim
colorscheme midnight-meadow
```

###### lua
```lua
vim.cmd.colorscheme("midnight-meadow")

-- or

vim.cmd[[colorscheme midnight-meadow]]
````
